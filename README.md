<h1>React Redux-Toolkit Blog</h1>

<b>Front-end</b> : React Redux Toolkit<br>
<b>Back-end</b> : Express Node MongoDB<br><br>

Contains all the expected functionality:<br>
View all posts.<br>
View posts by user.<br>
View single post.<br>
Like post.<br>
Create post.<br>
Edit post.<br>
Register and login.<br><br>
